About SwipeScript

Project Overview

SwipeScript is a full-stack portfolio project—a dating app clone I built to sharpen and showcase my skills in modern web development, secure architecture, and user-centered design.

Technology Stack

Frontend

  • Angular 18 with standalone components
  • TypeScript for type-safe development
  • Bootstrap 5 for responsive design
  • NGX libraries for notifications, loading indicators, and UI components
  • Real-time messaging and reactive forms

Backend

  • Spring Boot 3 REST API
  • Spring Security with JWT authentication
  • PostgreSQL database with JPA/Hibernate
  • Cloud-based image storage with Cloudinary

Key Features

  • Secure user authentication
  • Real-time private messaging
  • Pagination, filtering, and responsive design

Deployment

Deployed on Heroku, SwipeScript integrates PostgreSQL for data persistence and Cloudinary for image storage, demonstrating production-ready application management and cloud service integration.

Learning Outcomes

  • Implemented security best practices
  • Worked with external APIs and cloud services
  • Designed scalable databases and managed state effectively
  • Applied clean code principles and robust error handling

Future Enhancements

SwipeScript serves as a platform to explore new ideas and stay up-to-date with Angular and web technologies.
