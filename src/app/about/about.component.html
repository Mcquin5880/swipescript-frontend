Project Overview

SwipeScript is a full-stack demonstration project that playfully presents itself as a dating platform for dogs. While the concept is fun, the technical implementation is serious - showcasing modern web development practices and enterprise-level architecture. This portfolio piece demonstrates my ability to build secure, scalable, and user-friendly applications.

Technical Implementation

Frontend Technologies

  • Angular 18 with standalone components and signals for state management
  • TypeScript for type-safe development
  • Bootstrap 5 for responsive design
  • Real-time messaging system
  • Reactive forms with custom validators
  • NGX libraries:
    • ngx-toastr for user notifications
    • ngx-spinner for loading states
    • ngx-bootstrap for enhanced UI components

Backend Architecture

  • Spring Boot 3 REST API
  • Spring Security with JWT authentication
  • JPA/Hibernate for database operations
  • PostgreSQL database
  • RESTful API design principles

Key Features

  • Secure user authentication and authorization
  • Real-time private messaging system
  • Cloud-based image management with Cloudinary
  • Responsive design for mobile and desktop
  • Pagination and filtering capabilities

Deployment

The application is deployed on Heroku's cloud platform, utilizing PostgreSQL for data persistence and Cloudinary for image management. This demonstrates my ability to work with cloud services and manage production deployments.

Learning Outcomes

This project has been an excellent opportunity to:

  • Implement security best practices in both frontend and backend
  • Work with external APIs and cloud services
  • Handle state management in a modern web application
  • Design and implement a scalable database schema
  • Practice clean code principles and proper error handling

Future Enhancements

While this project serves primarily as a portfolio piece, I continue to use it as a playground for learning and implementing new technologies and best practices. Future updates may include enhanced real-time features, improved performance optimizations, and integration of new Angular features.
